# Importing these into Gamma

These files are formatted for **Gamma → Import → Paste in text** (or "Import from document"). Each file is one deliverable, already split into presentation **cards**.

## How to use
1. In Gamma, click **New → Import → Paste in text** (or **Create new → Paste in**).
2. When asked how to split, choose **"Split by `---`"** (each `---` becomes a new card).
3. Paste the contents of **one** `*.gamma.md` file at a time (or paste them in sequence to build one deck).
4. Pick a theme; let Gamma auto-generate layouts, then refine.

## Formatting conventions used
- `#` = deck / opening title card.
- `---` = card break (one card per section).
- `##` = card title; bullets = card body (kept short so Gamma can design them).
- Tables are used only where they read well as a card (backlog, assumptions, workshops).
- `> quote` blocks = pull-quote / highlight cards.

## Suggested order for a single combined deck
1. `02_Solution_Vision.gamma.md` (the backbone — context → vision → onboarding → delivery)
2. `03_Backlog.gamma.md` (MVP scope & release plan)
3. `04_Assumptions.gamma.md`
4. `05_Discovery_Workshops.gamma.md`
5. `06_Prototype_and_Close.gamma.md`
6. `01_Clarification_Questions.gamma.md` — use as an appendix, or fold the summary card near the front.

> Tip: for a tighter presale deck, present Vision + Backlog + Prototype live, and keep Clarifications / full Assumptions / Workshops as appendix cards.
